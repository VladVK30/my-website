# Калькулятор сна

Это веб-приложение помогает рассчитать оптимальное время для отхода ко сну на основе желаемого времени пробуждения. Приложение учитывает циклы сна (около 90 минут каждый) и предлагает несколько вариантов времени, когда следует лечь спать, чтобы проснуться в конце цикла и чувствовать себя более отдохнувшим.

## Особенности

- Простой и интуитивно понятный интерфейс
- Расчет оптимального времени сна на основе 4-6 циклов (по 90 минут)
- Возможность использовать текущее время как время пробуждения
- Адаптивный дизайн, работающий на различных устройствах
- Информация о циклах сна и советы для лучшего сна
- Современный UI с градиентным фоном и анимациями

## Как использовать

1. Введите желаемое время пробуждения или нажмите кнопку "Сейчас", чтобы использовать текущее время
2. Нажмите кнопку "Рассчитать время сна"
3. Выберите подходящий вариант времени сна из предложенных
4. Следуйте советам для улучшения качества сна

## Технические детали

- Использует Tailwind CSS для стилизации
- Использует Font Awesome для иконок
- Чистый JavaScript без использования фреймворков
- Адаптивный дизайн для мобильных устройств и десктопов
